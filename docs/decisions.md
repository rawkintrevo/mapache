# Decision Records

## Purpose

This page indexes accepted architecture decision records. The ADR bodies remain in `adrs/` so the decision archive stays separate from active wiki pages.

## Read When

Read this when changing GitHub App permissions, repository scope, PR behavior, branch naming, or a subsystem page links to a decision.

Agents need credentials, but credentials should not become ambient machine
state. The Authentication Center keeps that distinction visible.

{/* truncate */}

Store provider credentials in your profile, then enable them per workspace. The
runner receives generated files and environment values only for the providers
that workspace can use.

This is intentionally more explicit than a global `.env` file. If a project
does not need a provider, the session should not inherit it.

# Verify and Deploy

Every non-trivial change should leave a clear verification trail. Run checks
inside the relevant package before handing off or deploying.

## Frontend Build

Run the app build from the repository root:

```bash
npm run build
```

## Community Build

Run the community site build from the Docusaurus package:

```bash
cd community
npm run build
```

## Cloud Resources

This repository deploys to the `pi-agents-cloud` Firebase and GCP project. Use
explicit project flags on remote commands so local gcloud defaults cannot deploy
to the wrong project.

```bash
gcloud builds submit session-runner --project pi-agents-cloud --tag us-central1-docker.pkg.dev/pi-agents-cloud/pi-agents/session-runner:latest
```

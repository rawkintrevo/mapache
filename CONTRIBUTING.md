# Contributing

Thanks for your interest in the project.

This project is not accepting external code contributions at this time. Please do not open pull requests or submit patches unless explicitly requested by the maintainer.

Issues and discussions are open for bug reports, security concerns, questions, and general feedback. Please do not include proprietary code, secrets, credentials, or confidential information in public issues or discussions.

Ideas and feedback shared in issues or discussions may be used to improve the project, but copyrightable contributions such as code, documentation, or substantial implementation materials are not being accepted at this time.
